| Concept           | Description                                                                     | Example                           |
| ----------------- | ------------------------------------------------------------------------------- | --------------------------------- |
| `display: flex`   | Turns container into flex formatting context; direct children become flex items | `display:flex;`                   |
| `flex-direction`  | Sets main axis direction (`row`/`column`/`*-reverse`)                           | `flex-direction: column;`         |
| `justify-content` | Distributes leftover space along main axis (`center`, `space-between`)          | `justify-content: space-between;` |
| `align-items`     | Aligns items along cross axis (`stretch`, `center`, `baseline`)                 | `align-items: center;`            |
| `gap`             | Spacing between flex items (better than child margins)                          | `gap: 1rem;`                      |
